# Hochzeits-Fotobox

Die **Hochzeits-Fotobox** ist ein lokales System zur Aufnahme und Bereitstellung von Eventfotos mit QR-Galerie, Druck und Download. Die Lösung ist für den Offline- oder lokalen Netzwerkbetrieb ausgelegt und fokussiert einen einfachen Ablauf für Gäste und Veranstalter. Bilder werden automatisiert importiert, in einer Webgalerie angezeigt und optional in eine Druckwarteschlange übergeben.

## Features
- Lokale Fotobox-Nutzung für Veranstaltungen (z. B. Hochzeit)
- QR-Zugang zur Webgalerie für Gäste
- Bild-Download über die Galerie
- Übergabe von Bildern an eine Druckqueue
- Klarer, modularer Datenfluss zwischen Aufnahme, Import und Ausgabe

## Systemüberblick
Kamera -> Importdienst -> Webgalerie -> Druckqueue

## Datenschutz (Kurzkonzept)
- Betrieb primär lokal, ohne externe Cloud-Anbindung als Standard
